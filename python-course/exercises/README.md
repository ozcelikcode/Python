# Exercises

Bu klasor alistirma yapmak icindir.

Kullanma sekli:

1. Ilgili konuya ait egzersiz dosyasini ac.
2. Once kendi denemeni yaz.
3. Takildigin satiri bana gonder.
4. Ben sana direkt cevabi vermek yerine ipucu veririm.
5. Gerekirse birlikte satir satir duzeltiriz.

Kural:

- Buradaki dosyalar ders notu gibi degil, pratik alanidir.
- Hedef hizli bitirmek degil, anlamak ve tekrar etmektir.
